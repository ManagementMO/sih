# 2016.exe

A standalone cursed retro productivity app for the Stupid Ideas Hackathon.

## Run

From this folder:

```bash
python3 -m http.server 4181
```

Then open:

```text
http://localhost:4181
```

## What it does

- Turns a normal task into an absurd 2016 startup concept.
- Adds toxic Clippy VC advice.
- Inflates fake metrics and investor sentiment.
- Generates a LinkedIn humblebrag post.
- Plays browser-generated startup noises for key actions.

## Demo flow

1. Click `ENTER FOUNDER MODE`.
2. Type `buy milk` or use a preset.
3. Hit `DISRUPT TASK`.
4. Smash `ADD AI`, `MAKE IT B2B`, and `RAISE $12M SEED`.
5. Read the LinkedIn post out loud like you mean it.

## Notes

- Sound arms on first click to satisfy browser autoplay rules.
- `COPY PITCH` and `COPY POST` both include clipboard fallbacks.
- The whole project is static HTML, CSS, and JS with no build step.
