# 2016.exe

A standalone static joke app for the Stupid Ideas Hackathon.

It takes any normal task and converts it into:

- a fake 2016 startup keynote headline
- a bogus investor memo
- fake growth metrics
- a LinkedIn humblebrag post
- toxic Clippy-style startup coaching
- optional synthesized airhorn / dubstep / monetization sounds

## Run

From this folder:

```bash
python3 -m http.server 4175
```

Then open:

```text
http://localhost:4175
```

## Demo path

1. Click `Launch Founder Mode`
2. Leave `buy milk` or hit `Auto Demo`
3. Use `Go Public` for the loudest joke
4. Read the LinkedIn post out loud
5. Hit `Airhorn` at the worst possible moment
