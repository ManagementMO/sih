# 2016.exe Plan

## Visual Thesis

Make it feel like a cursed 2016 founder desktop: chrome gradients, giant WordArt, glowing glass panels, and absolute confidence detached from reality.

## Content Plan

1. Launch overlay
   One giant joke, one CTA, instant tone.
2. Main generator window
   Input a plain task and instantly overbrand it.
3. Dashboard windows
   Fake metrics, chart theater, investor sentiment, and a live hype feed.
4. Coach + brag layer
   Toxic Clippy advice, LinkedIn post generator, and copy buttons.

## Interaction Thesis

1. Boot transition
   The overlay drops you into a fake desktop.
2. Escalation buttons
   Each click makes the idea worse in a more specific way: pivot, AI, B2B, seed round, grindset.
3. Audio punctures
   Airhorn and dubstep reward the dumbest possible actions.

## Demo Safeguards

- A default sample task is already present.
- Preset chips give you fast fallback examples.
- Sound only arms after a click, so autoplay restrictions do not kill the demo.
- Copy actions fall back cleanly if clipboard permissions are blocked.
- The whole project is static and dependency-free.
